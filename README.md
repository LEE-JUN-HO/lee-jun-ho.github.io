## 🚀 GitHub Pages 5분 배포 가이드

준호님은 이미 `LEE-JUN-HO` GitHub 계정이 있으시니, 사용자명 페이지(`https://lee-jun-ho.github.io`)에 자동 배포됩니다.

### 1단계 — 새 리포 만들기
- 사이트에서 **New repository** 클릭
- **Repository name**: 정확히 `LEE-JUN-HO.github.io` 입력
- **Public** 체크
- **Create repository**

### 2단계 — 파일 업로드
리포에서 **Add file → Upload files** → 아래 3개 파일을 끌어다 놓기:
```
index.html
.nojekyll
README.md
```
- ⚠️ 폴더째 업로드 X. **3개 파일만 루트에**.
- **Commit changes** 클릭.

### 3단계 — Pages 활성화
리포 상단 **Settings → Pages**:
- **Source**: `Deploy from a branch`
- **Branch**: `main` / `/ (root)`
- **Save** 클릭

### 4단계 — 1~2분 기다리기
상단 빌드 로그가 끝나면 초록 박스에 URL:
> ✅ Your site is live at `https://lee-jun-ho.github.io/`

### 5단계 — 확인
브라우저에서 위 주소 이동. 안 보이면 `Ctrl + F5` (캐시 이슈).
